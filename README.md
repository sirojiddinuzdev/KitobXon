# KitobXon

Kitob almashtirish uchun platforma. Foydalanuvchilar o'qib bo'lgan kitoblarini boshqalar bilan almashishi mumkin.

## Ishlatilgan texnologiyalar

- Django
- PostgreSQL
- Redis
- Celery
- Docker
- Nginx
- AWS EC2

## Funksiyalar

- Kitob qo'shish va ko'rish
- Kitob qidirish
- Janr va hudud bo'yicha filter
- Kitob so'rovi yuborish
- So'rovni qabul yoki rad qilish
- Foydalanuvchi profili
- Email xabarnoma
- REST API va Swagger
